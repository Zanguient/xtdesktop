Release Notes
xTuple Desktop (xtDesktop)
Version 4.0.0
January 2015
==================================

This is the release of the 4.0.0 xTuple Desktop package,
a free and open source extension package for xTuple ERP (PostBooks, 
Standard, and Manufacturing Editions). 

xtDesktop is available under the same Common Public Attribution 
License (CPAL) as the PostBooks Edition of xTuple ERP 
(www.xtuple.com/CPAL). For more information on xTuple's extension 
package architecture, please see www.xtuple.org/developer-zone.

==================================

Significant Features:

Release 4.0.0 of the xTuple Desktop adds an updated look to the xTuple ERP
desktop client:
- The column of icons on the left-hand side of the main application window
  has been replaced with a text list of available panes.
- A user-configurable set of frequently used windows has been added to the
  left-hand side of the main window. This list contains the windows mapped
  to the function keys F1 through F5.
- The colors and visual style of the application as a whole has been changed.
  There is an option in User Preferences to use the default application style.

Features:

* Apply a new visual style to the desktop and application
  (xt-desktop revamp) [25211]
* Configurable Shortcuts page [25085]

Bug Fixes:

* Desktop Widget, Sales Order "Amount" field incorrect [24836]
